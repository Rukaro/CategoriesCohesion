类目内聚度分析 API 运行中

使用 SBERT 模型计算类目内聚度指标

可用接口

GET /api/health

健康检查接口，用于验证服务是否正常运行

curl http://localhost:5000/api/health
POST /api/calculate-cohesion

计算类目内聚度指标

请求体：

{
  "category": "水果",
  "items": ["苹果", "香蕉", "橙子", "葡萄"],
  "aggregation_method": "mean"  // 或 "median"
}

响应示例：

{
  "cohesion_score": 0.6599,
  "similarities": [0.7003, 0.6058, 0.6439, 0.6898],
  "method": "mean",
  "category": "水果",
  "items_count": 4
}

使用示例：

curl -X POST http://localhost:5000/api/calculate-cohesion \
  -H "Content-Type: application/json" \
  -d '{"category":"水果","items":["苹果","香蕉"],"aggregation_method":"mean"}'

说明

  • cohesion_score: 类目内聚度评分（0-1之间，越接近1表示关联性越强）
  • similarities: 每个普通词与类目词的相似度数组
  • aggregation_method: 聚合方法，支持 "mean"（平均值）或 "median"（中位数）

错误处理

访问不存在的路径会返回 404 错误，包含可用接口列表。
